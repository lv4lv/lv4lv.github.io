格式

标准格式为：

{{< shortcode parameter >}}
内容不被markdown解析                   
{{< /shortcode >}}

{{% shortcode parameter %}}
内容被markdown解析
{{% /shortcode %}}

{{< shortcode para1 para2 >}}
